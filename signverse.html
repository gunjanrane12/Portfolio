SignVerse

Indian Hand Sign Language Translator

Project Overview

  • Created an AI-based Indian hand sign language (IHSL) translator using image processing and deep learning to recognize and interpret letters and digits of Indian hand signs.
  • Processed webcam input in real-time to detect hand gestures using OpenCV and TensorFlow CNN models.
  • Introduced a learning module to help people learn sign language by interactive visuals and feedback.
  • Delivered the system through Flask with a user-friendly frontend for use as an educational or assistive tool.
  • Tech Stack: Python, TensorFlow, OpenCV, Flask, HTML, CSS, JavaScript
